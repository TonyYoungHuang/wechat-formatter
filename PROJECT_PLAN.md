# 微信公众号排版工具 - 项目说明

## 当前目标

构建一个面向微信公众号创作者的轻量排版站点，核心流程：

1. 导入原始文章（粘贴 / 上传）。
2. 智能一键排版（自动识别结构与重点）。
3. 生成公众号可粘贴的富文本 HTML。

## 已完成能力

- 浅色简洁双栏 UI，支持编辑区与手机预览区实时联动。
- 排版模板切换（薄荷、雾灰、暖杏、海盐）。
- 本地智能解析：自动识别标题、小标题、列表、序号、引用。
- 行内样式支持：`**加粗**`、`__下划线__`、`==高亮==`、`{green|颜色}`。
- 一键复制富文本（优先 `text/html`，降级为纯文本复制）。

## 后续建议

1. 接入真实 AI 模型配置页（支持 API Key 与模型选择）。
2. 增加历史草稿与模板管理。
3. 支持图片插入、封面生成与图文导出。
